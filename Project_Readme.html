Evolent Contact Manager

Please follow the below steps to run the solution :

  • 1. Setup Database in Microsoft SQL


        1.1 I have provided the sql scripts to create the database in Sql Server. Running the scripts will create tables and sample data. I have given my database name as "Evolent". In this project we will only be working with "Contacts" table to perform CURD operations using Web API and Entity framework.

  • 2. Build the Web API application


        2.1 Clone the application from GIT repository and after it is successful, build the application. It may have some missing dll's. I have put all the required dll's i.e. for respository, AutoMapper, BootStrap, etc in "packages" folder in the root directory.
        2.2 Build the application "Evolent.ContactManager" so that it gets compliled successfully.

  • 3. Host the Web API application


        3.1 Run the application "Evolent.ContactManager" or it can be hosted in IIS.

  • 4. Build the Client application


        4.1 Clone the application "Evolent.Client" from GIT repository and after it is successful, build the application.
        4.2 Build the application "Evolent.Client" so that it gets compliled successfully.

  • 4. Run the Client application


        4.1 Copy the URL of "Evolent.ContactManager" and replace the URL present in Web.Config of "Contact.Evolent" with key name "ContactAPI" in "appSettings" with the URL of Host Application
        4.2 Run the client application "Evolent.Client"
